Privacy Policy – URL Locale Cleaner

URL Locale Cleaner does not collect, store, transmit, or sell any personal data.

The extension operates entirely within the user’s browser. Its sole purpose is to clean language segments and tracking parameters from URLs when explicitly triggered by the user (e.g., clicking the extension button or using the context menu).

Data Handling

No personal information is collected.

No browsing history is stored or transmitted.

No data is sent to external servers.

No analytics, tracking, or telemetry is used.

All processing occurs locally in the browser session.

Permissions Usage

Permissions requested by the extension are used strictly to:

Access the active tab when the user activates the extension

Modify or read the current page URL to clean it

Copy the cleaned URL to the clipboard upon user request

These permissions are used only in response to explicit user interaction.

Third Parties

The extension does not use third-party services, external APIs, or remote code.
